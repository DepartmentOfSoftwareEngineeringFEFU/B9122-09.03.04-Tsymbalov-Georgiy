# B9122-09.03.04-Tsymbalov-Georgiy
Разработка системы мониторинга и предсказания аварийных ситуаций в электроэнергетической сети

Для получение реальных погодных данных нужно получить API ключ OpenWeatherMap, это можно сделать на их официальном сайте бесплатно, зарегестрировав свой аккаунт.
Его нужно вставить в файл ".env" в поле "OPENWEATHER_API_KEY"

для запуска проекта необходимо ввести команду в терминале "docker-compose up -d --build"

Данные для входа в акканут админа: admin | admin123 (логин | пароль)
Данные для входа в аккаунт диспетчера: user | user
